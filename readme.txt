==============================================================================
クリップボードの数値部分を増やしながら貼り付け
==============================================================================

クリップボーに格納されている文字列
	dst[0]=src[n+0xfe]+0xDEAD+0XBEEF;

貼り付け
１回目：dst[1]=src[n+0xff]+0xDEAE+0XBEF0;
２回目：dst[2]=src[n+0x100]+0xDEAF+0XBEF1;
３回目：dst[3]=src[n+0x101]+0xDEB0+0XBEF2;
４回目：dst[4]=src[n+0x102]+0xDEB1+0XBEF3;
